Here are all the ways to mention a certain entity within discord.

## Users

You can use angle brackets `<>`, `@`, and the users id.

<Markdown 
  client:load 
  value="<@378293909610037252>" 
  discordCallback={{ user: "@Hacksore" }}
/>

## Roles

You can use angle brackets `<>`, `@&`, and the roles id.

<Markdown 
  client:load 
  value="<@&378293909610037252>" 
  discordCallback={{ role: "@Mods" }}
/>

## Channels

You can use angle brackets `<>`, `#`, and the channel id.

<Markdown 
  client:load 
  value="<#378293909610037252>" 
  discordCallback={{ channel: "#some-cool-channel" }}
/>

## Emojis

You can use angle brackets `<>`, `:` and emoji `name` : `id` to show up the emoji.

<Markdown 
  client:load 
  value="<:overlayed:1263836954789806154>"
  discordCallback={{ emoji: "ðŸ’Ž" }}
/>
